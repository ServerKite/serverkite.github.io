bench.sh

Server Benchmark Script

A stupidly-simple server benchmark script written to benchmark your VPS and/or Dedicated machine. Handles speed test (100MB test file size), Disk I/O, Ping & Traceroute tests.
So, do you dare to put your machine to the test?

git clone https://github.com/ServerKite/bench.sh.git && cd bench.sh && chmod +x bench.sh && bash bench.sh
