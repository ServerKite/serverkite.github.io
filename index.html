· Benchmark Your Linux Machine ·

ServerKite is a stupidly-simple server benchmark script written to benchmark your Virtual Private Server and/or Dedicated Server. Handles speed, I/O, Ping tests etc. — Put your machine to the test and see if it is up to standard, in terms of performance in accordance to it's 'on-paper' specifications.

git clone https://github.com/ServerKite/bench.sh.git && cd bench.sh && chmod +x bench.sh && bash bench.sh
