ServerKite

A simple, server benchmark script written to benchmark your Virtual Private Server and/or Dedicated Server Server via a Bash script that does speed test, I/O and UnixBench test.

FEATURES

Speed Test - I/O - IOPing - UnixBench

  • Speed tests from 30+ well-known DataCenters and +10 top VPS providers across the World
  • Over 50+ speed test locations including: North America, South America, Europe, Asia & Middle East
  • Verity of 10MB, 100MB & 1GB test file sizes included; 100MB for the most
  • Pings, seek test, sequential, cache and non-cache I/O tests
  • Several dd tests and raw FIO test
  • Speedtest.net test w/ link
  • Ping, traceroute and a lot more!

Using ServerKite:

Download the bench.sh via the following link:

Using GNU Wget:

wget http://serverkite.com/bench.sh

Using Aget:

aget http://serverkite.com/bench.sh

Next, you'll have to make the bash file executable. You can do so by running this commands:

chmod +x bench.sh

Once you have downloaded the script onto your server, all that's left to do is execute the bench.sh file using the bash commands:

bash bench.sh

or alternativly

./bench.sh

Now, grab a cup of coffee or a can of beer and wait. The estimated time to finish the whole process is 10 - 15 Minutes, depending on the network.
