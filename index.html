ServerKite

A stupidly-simple server benchmark script written to benchmark your Virtual Private Server and/or Dedicated Server. Handles speed, I/O, Ping tests etc.

Fork me on GitHub @ServerKite

FEATURES

Speed Test · I/O · Ping · Traceroute

  • Speed tests from 30+ well-known data centers across the World
  • Speed tests from +10 VPS providers across the World
  • 100+ speed test links in various locations including: North/South America, Europe, Asia & Middle East
  • Standard across the board with 100MB size speed test links
  • Pings, seek test, sequential, cache and non-cache I/O tests
  • Several dd tests and raw FIO test
  • Speedtest.net, ping, traceroute and a lot more!

Using bench.sh:

We recommend that you download via "Git". — On Debian, you dowload it via the 'apt-get' repo by executing apt-get install git-core

Once Git is installed on your machine, execute the following command:

git clone https://github.com/ServerKite/bench.sh.git && cd bench.sh && chmod +x bench.sh && bash bench.sh

[Alternative]: In support of vpsBoard, we have deployed a repo there hosted using GitLab.

git clone http://git.vpsboard.com/ServerKite/bench-sh.git && cd bench-sh && chmod +x bench.sh && bash bench.sh

Now, grab a cup of coffee or a can of beer and wait. The estimated time to finish the whole process is 10 - 15 Minutes — then again, depending on the network.
